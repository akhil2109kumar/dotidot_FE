# Setup Details

# Prerequisites
 - node version : v16.13.0 (min.)

## Used Technologies

- React
- React Flow

## setup
```
> git clone https://github.com/akhil2109kumar/dotidot_FE.git
> cd dotidot_FE

Run :-
> npm install
> npm run dev

```
After running the start command the server will start on localhost:5173
